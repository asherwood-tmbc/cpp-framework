# CPP.Framework.Web

Provides helper classes for accessing web technologies, such as REST API services.

## Features

- **HTTP Service Client** — Strongly-typed REST client with error handling and request formatting
- **Security Attributes** — `SecurityAuthorizeAttribute` and `SecurityAllowAnonymousAttribute` for MVC and Web API controllers
- **Parameter Formatters** — HTTP, WCF, and API service parameter formatting helpers
