# CPP.Framework.Core

Provides basic application services, such as method parameter validation, IoC injection containers, reading application configuration settings, etc.

## Features

- **Dependency Injection** — Unity-based service locator and container integration
- **Configuration** — Bridged access to application configuration settings via `ConfigSettingProvider`
- **Diagnostics** — Journaling and telemetry infrastructure with pluggable listeners
- **Cryptography** — Certificate-based encryption and decryption services
- **Security** — Claims-based authorization policies and permission management
- **Validation** — Extended data annotation validators including dependent and conditional validation
- **Collections** — Specialized collections such as `SetDictionary`, `StateChangeMap`, and contravariant collections
- **Messaging** — Mail message abstractions and attachment resolution
- **Threading** — Async locks, thread limiters, and cancellable resource management
