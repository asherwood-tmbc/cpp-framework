# CPP.Framework.WindowsAzure.ApplicationInsights

Provides Application Insights integration for the CPP Windows Azure framework.

## Features

- **Journal Listener** — Routes CPP framework journal events to Application Insights as traces and exceptions
- **App Insights Provider** — Configurable `TelemetryClient` wrapper with support for custom telemetry configuration
- **Configuration Extensions** — Helper extensions for wiring Application Insights into the CPP configuration provider
