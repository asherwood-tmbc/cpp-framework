# CPP.Framework.WindowsAzure

Provides helper classes to simplify access to Azure Storage, Service Bus, and the WebJob SDK.

## Features

- **Azure Service Bus** — Topic, subscription, and queue abstractions with typed message property support
- **Azure Storage** — Blob, table, queue, and lease management wrappers with retry policies and filter builders
- **Azure WebJobs** — WebJob host integration with journal listener and Service Bus trigger support
- **Configuration** — Azure Cloud Services role environment and cloud configuration provider integration
- **Diagnostics** — Journal listeners for Azure Storage and WebJob-hosted scenarios
- **Messaging** — Azure Blob-backed mail attachment provider
