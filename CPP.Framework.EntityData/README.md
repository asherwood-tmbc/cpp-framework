# CPP.Framework.EntityData

Provides helper classes to simplify access to Entity Framework database first components.

## Features

- **EF Data Source Filter** — Extends `DataSourceFilter` with Entity Framework 6 query integration
- **Entity Exceptions** — Typed exceptions for missing entities, invalid relations, and data entity errors
- **Entity Relations** — Helpers for navigating and resolving Entity Framework entity relationships
- **Diagnostics** — Journal source extensions for Entity Framework operations
